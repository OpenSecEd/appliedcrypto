### Admin

[Register
](https://www.kth.se/en/student/kurs/kursregistrering-1.317058){.external
target="_blank"}for the course.

Review your[ Canvas notification settings
](https://guides.instructure.com/m/4212/l/710344-how-do-i-set-my-canvas-notification-preferences-as-a-student){.external
target="_blank"}(Course-specific [tutorial
here](https://www.iorad.com/player/2244141/Canvas-KTH---How-to-set-your-notifications-preferences-for-discussions-and-announcements){.inline_disabled
target="_blank"}) to match your needs. Ensure that Announcement is set
to \"Notify me right away\" as this is how we update you on urgent
changes in the course. Same for submission comments on assignments.

### Funka, accessibility

If you have any disability, you can get support through [Funka.
](https://www.kth.se/en/student/studentliv/funktionsnedsattning/funka-stod-for-studenter-med-funktionsnedsattningar-1.953214){.external
target="_blank"}For any requirements for accommodation, contact the
course leader \[Daniel Bosk \<dbosk@kth.se\>\] using a good subject line
(tilkryYY funka accomodation). The same goes for if you have suggestions
on how to make the course more accessible.
